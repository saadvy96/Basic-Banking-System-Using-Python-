# Basic-Banking-System-Using-Python-

🏦 Python Banking System

A simple command-line based banking system built using Python. This project simulates basic banking operations like account creation, deposits, withdrawals, and account management using file handling for data persistence.

🚀 Features

✅ Create a new bank account

💰 Deposit money with limit checks

💸 Withdraw money with balance validation

🔍 View account details securely (PIN protected)

✏️ Update account information

❌ Delete account with confirmation

💾 Data stored persistently using JSON file

🛠️ Tech Stack

Python 🐍

JSON (for database storage)

File Handling

OOP Concepts (Classes & Methods)

📂 Project Structure
├── data.json        # Stores all account data
├── main.py          # Main banking system code
